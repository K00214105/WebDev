Introduction

This website was created as part of a college project. I was tasked with creating a responsive web page to house a informational video. This video is to help the Health Service Execitive (West) raise awareness of the serious health issue that is VTEC. The target audience is English speaking users in the 13+ age group. I decided to create an infographic type video that features animation and animated text mainly. Imagery and video will also be used to convey the message. I researched by looking at some similar creations to see what has already been designed and this helped me to get some ideas for my video. I created a list of the features that I liked in the videos I researched.

Stage one - Audience Analysis

The target audience is English speaking users in the 13+ age group. I created two personas documenting assumptions of the target audience. The personas are of future viewers of the website and video.The other two are of people who are interested in learning about the VTEC virus.

Stage two - Mood Board

I decided to use the same colour scheme as the HSE website to keep consistency. The colour scheme works very well, and I like the simplicity of the website. I then took a screen capture of the HSE website and imported the image into Abobe Color to create the colour palette I planned on using. I then used a feature in Adobe Color that allowed me to get the exact colours from the screen capture of the HSE website to create my colour palette. This is a very useful tool and you can save your palettes to use in the future. Stage Four -Construction

I created a local root folder for the website and began putting in all of the information and content I had prepared. The website is fully responsive so it looks good on all sizes of devices. This was done using media queries and changing the attributes of certain elements for different screen sizes. I used my prototypes and moodboard as a guide to create the website. I provided some links to more information on VTEC and HUS and also links to the HSE social media outlets in the footer section of the page. The video is embeded using an iframe tag and the youtube link to the video.
